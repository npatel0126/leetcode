# Hashing – LeetCode Solutions

This folder contains my solutions to hashing-related problems from LeetCode.  
The focus is on using hash maps and hash sets to improve efficiency, optimize lookups, and solve problems involving frequency counting, uniqueness, and quick value retrieval.

## Topics Covered
- Hash Maps / Dictionaries
- Hash Sets
- Frequency Counting
- Counting Occurrences
- Duplicate Detection
- Two Sum Pattern
- Grouping and Mapping Values
- Prefix Sum + Hash Map
- Fast Lookup Problems
- Optimized Time Complexity
- Edge Case Handling
